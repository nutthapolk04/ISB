# capacitor-hardware

plugin for kiosk with nk77 bill acceptor

## Install

To use npm

```bash
npm install capacitor-hardware
````

To use yarn

```bash
yarn add capacitor-hardware
```

Sync native files

```bash
npx cap sync
```
